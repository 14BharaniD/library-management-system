A complete full-stack web application built from scratch using **Spring Boot**, **MySQL**, and **Thymeleaf**. This system provides a simple, clean, and responsive user interface for a librarian to manage a book inventory.

This project demonstrates a 3-tier architecture (View-Controller-Repository) and follows modern Java development practices.

## ‚ú® Features

* **View All Books:** A clean, responsive table that displays all books from the database.
* **Add New Books:** A simple form to add new books (title, author, ISBN) directly to the inventory.
* **Database Persistence:** All data is stored and managed in a **MySQL** database, so book information is never lost on restart.
* **RESTful API Backend:** The frontend communicates with a robust backend API built with Spring Boot.
* **Dynamic Frontend:** The frontend is rendered dynamically using **Thymeleaf** to display data from the backend.
